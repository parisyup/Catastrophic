This containts the final build of the game.

To play the game just launch the Catastrophic.exe file.

To create a network and play with other players. Log me in hamatchi needs to be installed. After installing hamatchi create a network in hamatchi and invite the other players to the network
Afterwards, one player needs to click on host in the game and the other player needs to click join and get the ip from hamatchi.
To get the ip right click the hosts name on hamatchi and copy address. Paste that address into the join screen and click join.

you should be able to connect to the game then.

original game rules can be found here: https://catastrophic.york.ac.uk/rules
